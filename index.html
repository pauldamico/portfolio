Projects & Apps

WOW Character Lookup

Currently in development, the system performs multiple API calls across all World of Warcraft servers to search for specific character names. It retrieves comprehensive character data, including stats, gear, and talents. Additionally, it features an easy-to-use interface for copying character talent configurations. The search functionality includes a debounce mechanism, optimized with a 1000 ms delay to improve performance and reduce unnecessary requests.

TECHNOLOGIES: React Native, Node, Express, Mongoose, Firebase(handles users/tokens), App deployed with EAS through EXO
